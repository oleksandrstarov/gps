
- copy files and folders as they are to yout web site. Php support is required.
- open config.php and set passwords for admin, live tracking and gpx uploads.
- data gets stored to folder defined in config.php. You can change the folder name and location. All the data is accessed via scripts so that folder does not need to shared over web.
- admin.php is tool for setting up tracking and archiving sessions. You can rename it as you wish.
- gps.php is used by phone tarcking client apps. You can rename it to hide it if you wish.
